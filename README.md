（基于ThinkPHP 6.0）毕业设计项目
===============
> 运行环境要求PHP7.1+。

## 主要功能（第一版）

* 登录：验证码、登录验证中间件
* 用户管理
  * 管理员：增删改查
* 商品管理
  * 商品类型管理：增删改查
  * 商品管理：增删改查
  * 入库管理：增删查
  * 出库管理：增删查
* 退出

ThinkPHP 6.0
===============

## 主要新特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 原生多应用支持
* 更强大和易用的查询
* 全新的事件系统
* 模型事件和数据库事件统一纳入事件系统
* 模板引擎分离出核心
* 内部功能中间件化
* SESSION/Cookie机制改进
* 对Swoole以及协程支持改进
* 对IDE更加友好
* 统一和精简大量用法
